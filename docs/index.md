# Nginx Proxy Manager SDK for Python

## Overview

This package provides a modern, typed Python interface for the Nginx Proxy Manager API.

## Why this fork is modernized

- Pythonic resource clients for proxy hosts and certificates
- Python packaging via `pyproject.toml` and setuptools
- Automated linting with pre-commit, Black, and Flake8
- Unit tests with `pytest`
- API reference generated with MkDocs and mkdocstrings
- GitHub Actions for CI and PyPI publishing
